# О Базе Данных

База данных по квадратурным модуляторам, попадающим в диапазон частот от 100 МГц до 6 ГГц.

Исходная таблица, подлежащася конвертации, находится в файле "db1.xlsx".

В данной базе данных для сортировки и фильтрации вынесены следующие параметры:
 - Наименование компонента
 - Производитель
 - Диапазон, Hz
 - Напряжение питания, V
 - Вносимые потери, dB
 - Подавление несущего колебания, dBm
 - Подавление нежелательной (боковой) полосы БП, dBc
 - Потери преобразования, dB
 - Спектральная плотность шума, dBm/Hz
 - Точка перехвата третьего порядка (OIP3), dBm
 

# Работа с базой данных.

Открытие базы происходит после запуска программы "main.exe".

В пользовательском интерфейсе можно сортировать строки столбцов по алфавиту/возрастанию/убыванию путём нажатия по названию столбца правой кнопкой мыши.

Для сортировки значений надо ввести нужные параметры в соответствующие столбцу текстовые поля и нажать клавишу "Enter".

Сортировку можно сбросить, нажав кнопку "Отчистить".


# Дополнение базы данных.

Дополнить базу данных можно модификацией файла "db1.xlsx", путём добавления новых строк.

После модификации, базу нужно конвертировать. 


Подробнее про конвертацию, формат, работу с программой и её настройку в файле "README.txt".
